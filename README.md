PathMind is a real-time 3D A* pathfinding visualizer with terrain intelligence.
Local dev: install deps, run the Next.js app, and run the FastAPI backend.
Frontend: `npm run dev` (set `NEXT_PUBLIC_API_URL` to your API URL).
Backend: `uvicorn main:app --reload --port 8000` from `backend/`.
